# Graph Extraction

`src/graph` 负责从笔记正文中提取图相关基础信息，例如 wiki-links 和 tags。

## 适用范围

- 文本扫描与解析
- 链接、标签、节点/边基础数据提取
- 纯内容到结构化数据的转换

## 当前代码特征

- `extractor.ts` 目前包含 wiki-link 与 tag 的正则提取逻辑。
- 这一层只关心“从文本中提取什么”，不关心 renderer 如何布局或渲染图谱。

## 约束

- 保持纯函数风格：输入内容，输出结构化结果。
- 正则与解析逻辑要尽量可读、可测试，避免混入调用方上下文。
- 不要引入节点坐标、视图布局、交互状态等 renderer 概念。
- 当语法支持扩展时，优先在这里集中维护解析规则，而不是让上层各自重复实现。
