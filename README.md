# AIMO Note

AIMO Note 是一款本地优先（Local-first）的桌面笔记应用，适合用 Markdown 管理知识、文档和个人笔记。

它的目标是：**让你的笔记始终保存在本地，可控、可迁移、可长期使用**。

## 这是什么

AIMO Note 可以理解为一款 Obsidian 风格的本地知识库工具：

- 用文件夹作为你的笔记库（Vault）
- 用 `.md` 文件直接保存内容
- 支持双向链接（`[[Note]]`）建立知识网络
- 提供全文搜索与关系图谱视图

## 核心功能

- **本地优先**：笔记数据保存在本地目录，不依赖云端
- **Markdown 编辑**：专注写作与结构化记录
- **双向链接**：快速建立笔记之间的关联
- **关系图谱**：可视化查看知识连接
- **全文搜索**：快速定位内容
- **桌面应用体验**：独立窗口、原生菜单与系统托盘能力

## 适合谁

- 想搭建个人知识库的用户
- 希望长期使用 Markdown 管理笔记的人
- 关注数据可控、可备份、可迁移的团队或个人

## 快速开始

1. 启动 AIMO Note
2. 选择或创建一个本地笔记目录（Vault）
3. 新建 Markdown 笔记开始记录
4. 在笔记中使用 `[[笔记名]]` 建立关联
5. 通过搜索和图谱探索你的知识网络

## 数据与隐私

- 你的笔记文件默认保存在你选择的本地目录
- 你可以用任何支持 Markdown 的工具直接打开这些文件
- 建议你定期备份 Vault（例如 iCloud、OneDrive、NAS、Git）

## 常见问题

### 1) 可以导出数据吗？

AIMO Note 直接使用本地 Markdown 文件，无需特殊导出步骤；复制 Vault 目录即可迁移。

### 2) 是否支持离线使用？

支持。核心笔记能力可离线使用。

### 3) 支持哪些系统？

项目基于 Electron，通常可构建为 macOS / Windows / Linux 桌面应用。
